# leon-easy-om

###  批任务处理组件
#### 介绍


-   每个批处理任务挂载多个子任务，所有子任务执行完成，批处理任务视为执行完成； 
-   支持多机房场景下：所有机房执行任务、任意机房执行任务
-   支持批处理任务：上下游任务依赖
-   支持批任务执行器分组，每个执行器配置不同的线程池资源；
-   支持任务统计信息，任务中断、取消；
-   支持可扩展的机房路由，可扩展的任务容器工厂，可扩展的任务获取周期策略
-   支持批任务取消，中断操作
